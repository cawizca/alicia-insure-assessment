# Alicia Insure (GapStars) Assessment

A music discovery application built with React, TypeScript, and Chakra UI that integrates with the Last.fm API.

## Setup

1. **Install dependencies:**

   ```bash
   yarn install
   ```

2. **Configure environment variables:**
   - Copy `.env.example` to `.env`
   - Add your Last.fm API credentials:
     ```
     VITE_LASTFM_API_KEY=your_api_key_here
     VITE_LASTFM_BASE_URL=https://ws.audioscrobbler.com/2.0/
     ```

## Run

**Development:**

```bash
yarn dev
```

**Tests:**

```bash
yarn test
```

**Build:**

```bash
yarn build
```
